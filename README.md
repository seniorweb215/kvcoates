## Installation
1. Clone the project to your desired path `git clone https://rauxmedia@bitbucket.org/rauxmedia/api-digitalmedia-com.git path/to/your/folder`
2. Download [Composer](http://getcomposer.org/doc/00-intro.md) or update `composer self-update`.
3. Navigate to the folder containing the project.
4. Run `composer install`.

## Configuration

Read and edit `config/app.php` and setup the 'Datasources' and any other
configuration relevant for your application.
